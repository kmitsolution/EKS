# EKS
<ul>
  <li>Amazon Elastic Kubernetes Service (EKS) is a managed Kubernetes service that makes it easy for you to run Kubernetes on AWS and on-premises.</li>
  <li>Amazon EKS automatically manages the availability and scalability of the Kubernetes control plane </li>  
  <li>EKS lets you run your Kubernetes applications on both Amazon EC2 and AWS Fargate</li>
  <li>Amazon EKS supports using Elastic Load Balancing including Application Load Balancer (ALB), Network Load Balancer (NLB), and Classic Load Balancer.</li>
  <li>Amazon EKS is integrated with AWS CloudTrail to provide visibility and audit history of EKS management operations.</li>
  </u>
